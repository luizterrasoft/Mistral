How to use TtsqUnzip

 

Introduction

TtsqUnzip is a delphi freeware component used to decompress an archive generated by a PKZIP-compatible compressors such as winzip or PKZIP. The component is built on Theodoros Bebekis's translation(unzip32.pas) of Info-zip's unzip32.dll.

The key features of TtsqUnzip are:

  • free even for commercial purpose
  • very fast, same speed level as winzip since both the component and winzip uses the same c core code
  • mature and stable since it is build on Info-zip's implementations.
  • easy to use.
  • exposing all callbacks of the unzip32.dll. ready for customization.
  • working with the latest versions of unzip32.dll(v5.5)

Decompress a .zip file

In the following descriptions, we assume that we put a TtsqUnzip object on a form and the object name is tsqunzip1.

Unzip a .zip into a folder

tsqUnzip1.ZipFileName := 'c:\test_tsqzip.zip';
tsqUnZip1.ZipFileName := 'c:\tsqziptest.zip';
tsqUnZip1.DirToExtractTo := 'c:\mydir';
tsqUnZip1.UnZipOptions.DirCreationOption := dcoSafePath;
tsqUnZip1.Execute;

Show decompressing extra information

Use UnzipPrint, UnzipMessage, UnzipService event handlers.
